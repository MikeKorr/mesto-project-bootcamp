# Проект: Место

### Используеые технологии

- HTML5
- CSS3
- GRID
- FlexBox
- JavaScript

**Описание**

Данный проект позволяет каждому пользователю делиться своими фотографиями. Также пользователи могут ставить оценки авторам и их фотографиям.
